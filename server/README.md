# Great Chat Server

Server scaffold for the Great Chat application.

## Structure

- `src/app.ts` - Express application setup
- `src/server.ts` - HTTP server bootstrap
- `src/config/` - configuration utilities
- `src/modules/` - feature modules
- `src/middlewares/` - common Express middlewares
- `src/routes/` - route registration
- `src/sockets/` - socket.io event handling
- `src/utils/` - helper utilities
- `uploads/` - uploaded files
- `logs/` - logs

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
